# Full-Stack Form App

## Quick Start

```bash
# Clone and install
git clone https://github.com/vijayrampeesa59-ai/Docker-Assignment-5
cd project

# Run with Docker Compose
docker-compose up --build

# Access frontend at http://localhost:3000
# Backend API at http://localhost:5000
```

## Docker Hub Images
- Frontend: `vijayrampeesa/form-frontend:latest`
- Backend: `vijayrampeesa/form-backend:latest`
